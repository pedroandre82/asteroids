# Asteroids

A classic Asteroids clone built using Python and the Pygame library.

## Description

This project is a 2D space shooter where the player controls a ship and must avoid or destroy incoming asteroids. It demonstrates key game development concepts such as:
- Object-oriented programming (OOP) with class inheritance.
- Collision detection using circular hitboxes.
- Sprite grouping and management.
- A delta-time-based game loop for consistent movement speeds.

## Prerequisites

- Python 3.13
- Pygame

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/pedroandre82/asteroids.git
   cd asteroids
    ```

2. Install dependencies:
    ```bash
    pip install pygame
    ```

## How to Play
Run the main script to start the game:
```bash
python3 main.py
```

- Move: Arrow keys
